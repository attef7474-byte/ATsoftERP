## Common Tasks

### View Open Requests
1. Go to **Requests**
2. Filter by status = Open
3. View details of each

### View My Tasks
1. Go to **Tasks**
2. Click **My Tasks**
3. View tasks assigned to you

### Log Downtime
1. Go to **Downtime**
2. Click **Log Downtime**
3. Select machine, enter start time and reason

### Check Machine Details
1. Go to **Machines**
2. Click a machine to view its card
3. See status, location, warranty, documents

### View Upcoming Preventive
1. Go to **Preventive**
2. View upcoming/overdue schedules
3. Calendar view available
