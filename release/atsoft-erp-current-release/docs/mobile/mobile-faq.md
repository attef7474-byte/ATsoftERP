# Mobile FAQ

> Frequently asked questions about mobile access

## Can I access ATsoft ERP on my phone?

Yes, via the web browser. There is no native app yet.

## Does it work on iPhone and Android?

Both are supported via their respective browsers (Safari, Chrome).

## Can I scan barcodes with my phone camera?

Not in this release. Barcode scanning is a future enhancement.

## Can I receive push notifications on my phone?

Not in this release. Notifications are only available within the web app.

## Can I work offline?

No. A stable internet connection is required.

## Is there a tablet-optimized version?

The web interface is responsive and works reasonably well on tablets, but it is not fully optimized for touch input.

## When will a mobile app be available?

A mobile app (likely Flutter) is under consideration but no timeline is available.
