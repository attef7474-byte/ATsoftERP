# Mobile Guide — ATsoft ERP

> Mobile access overview

## Current Status

ATsoft ERP is a **web-based application** designed for desktop browsers. The interface is responsive and can be accessed from tablet browsers, but there is no native mobile app in this release.

## Access on Mobile

1. Open a web browser on your device
2. Go to your ATsoft ERP instance URL
3. Log in with your credentials

## Limitations

- The interface is optimized for desktop (1024px+)
- Some complex layouts (e.g., Tree view, large tables) may be difficult to use on small screens
- No native push notifications
- No offline mode
- No camera/barcode scanner integration

## Future

- A mobile-optimized interface is under consideration for a future release
- Native (Flutter) app is under consideration for a future release
