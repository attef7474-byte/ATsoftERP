# Inventory Team Quick Reference

> Daily operations for inventory users

## Access

| Page | Sidebar Path |
|------|--------------|
| Warehouses | Sidebar → Warehouses |
| Locations | Sidebar → Locations (under Inventory) |
| Products | Sidebar → Products |
| Product Categories | Sidebar → Product Categories |
| Inventory Balances | Sidebar → Inventory → Balances |
| Inventory Movements | Sidebar → Inventory → Movements |
| Inventory Counts | Sidebar → Inventory → Counts |

## Common Tasks

### Check Stock Level
1. Go to **Balances**
2. Search for product or location
3. View current quantity

### View Product Details
1. Go to **Products**
2. Click a product to see its details
3. View category, unit, status

### View Movement History
1. Go to **Movements**
2. Filter by product or date range
3. View transaction log

### Export Inventory Report
1. Go to **Reports** → Inventory section
2. Select report (Overview, Balances, Movements)
3. Click Export CSV or Export Excel
