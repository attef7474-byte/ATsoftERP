# Operator Quick Reference

> Daily operations for all users

## Access

| Task | How |
|------|-----|
| Open the system | http://localhost:3000 |
| Log in | Enter email + password, click Sign In |
| Switch language | Click language icon in top bar |
| Search | Press F9 or Ctrl+K |
| Log out | Click user menu → Logout |

## Navigation

- **Sidebar**: Shows all available modules
- **Dashboard**: First page after login
- **Search**: Find anything across all modules

## Quick Tasks

| Task | Path |
|------|------|
| View Dashboard | Sidebar → Dashboard |
| View Alerts | Sidebar → Alerts |
| View Notifications | Sidebar → Notifications or bell icon |
| View Reports | Sidebar → Reports |
| Search | F9 / Ctrl+K |

## Not Available

The following are **not** available:
- Sales, Purchasing, Finance, HR
- AI, IoT, BI, Forecasting
- Workflows, Import/Export Designer, Print Template Designer
