# Session 4: Barcodes, Reports, and Search

> Training session for all users — 1 hour

## Objectives

By the end of this session, you will be able to:
- View barcode and QR records
- Generate labels
- View operational reports
- Export to CSV and Excel
- Use the F9 search

## Barcodes and QR

1. Navigate to **Barcodes**
2. View barcode records
3. View available templates
4. Generate labels for products or machines

## Scanning

1. Navigate to **Scan** under Barcodes
2. Record a scan event
3. Choose scan type: inventory count, machine check, maintenance, part lookup
4. View scan history

## Reports

1. Navigate to **Reports**
2. Select a report category (Inventory, Maintenance, etc.)
3. Apply filters as needed
4. View the data

## Export

1. While viewing a report, click **Export CSV**
2. The file downloads and opens in Excel
3. Click **Export Excel** for .xlsx format

## Print

1. Click **Print** on a report
2. The browser print dialog opens
3. Choose a printer or **Save as PDF**

**Note**: PDF is browser print-to-PDF, not server-side generation.

## F9 Search

1. Press **F9** or **Ctrl+K**
2. Type any search term
3. Results appear grouped by module
4. Click to navigate

## Practical Exercise

1. Open Reports → find the Low Stock report
2. Export it as CSV
3. Open F9 search and find a product
4. Try the Barcode section

## Summary

You now know how to:
- Use barcodes and QR
- View and export reports
- Print to PDF
- Search across the system
