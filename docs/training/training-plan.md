# Training Plan

> Batch 39 — Training schedule and objectives

## Total Duration

7 hours (can be split across multiple days)

## Sessions

### Session 1: System Overview (1 hour)

**Audience**: All users

**Objectives**:
- Understand what ATsoft ERP does
- Log in successfully
- Navigate the sidebar and top bar
- Switch language
- Use the F9 search

**Agenda**:
1. What is ATsoft ERP? (10 min)
2. Logging in (10 min)
3. Dashboard overview (15 min)
4. Navigation and sidebar (15 min)
5. Language switch and F9 search (10 min)

### Session 2: Inventory Operations (2 hours)

**Audience**: Inventory team

**Objectives**:
- View warehouses and locations
- Browse products and categories
- Check inventory balances
- View movement history
- Understand inventory counts

**Agenda**:
1. Warehouses and locations (20 min)
2. Products and categories (20 min)
3. Inventory balances (20 min)
4. Movements (20 min)
5. Inventory counts overview (20 min)
6. Practical exercise (20 min)

### Session 3: Maintenance Operations (2 hours)

**Audience**: Maintenance team

**Objectives**:
- View machines and their details
- Understand maintenance dashboard
- Create and track requests
- Manage tasks
- Log downtime
- View preventive schedules

**Agenda**:
1. Machines, parts, documents (20 min)
2. Maintenance dashboard KPIs (20 min)
3. Requests (20 min)
4. Tasks and assignments (20 min)
5. Preventive and downtime (20 min)
6. Practical exercise (20 min)

### Session 4: Barcodes, Reports, Search (1 hour)

**Audience**: All users

**Objectives**:
- Understand barcode/QR functionality
- View and export reports
- Use unified search

**Agenda**:
1. Barcodes and QR labels (15 min)
2. Scanning (10 min)
3. Reports overview (15 min)
4. CSV and Excel export (10 min)
5. Print and PDF (5 min)
6. F9 search (5 min)

### Session 5: Admin, Security, Backup (2 hours)

**Audience**: Administrators

**Objectives**:
- Manage users, roles, permissions
- Understand security model
- Start/stop API and Web
- Run health checks
- Understand backup procedures

**Agenda**:
1. User management (20 min)
2. Roles and permissions (20 min)
3. Security model overview (20 min)
4. API/Web startup and monitoring (20 min)
5. Health and smoke checks (20 min)
6. Backup overview (20 min)

## After Training

- Distribute quick reference cards
- Complete the assessment checklist
- Provide access to the user manual
