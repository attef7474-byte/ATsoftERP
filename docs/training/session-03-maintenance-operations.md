# Session 3: Maintenance Operations

> Training session for maintenance team — 2 hours

## Objectives

By the end of this session, you will be able to:
- View machines and their details
- Understand the maintenance dashboard
- View maintenance requests
- View and manage tasks
- Log downtime
- View preventive schedules

## Machines

1. Navigate to **Machines** under Maintenance
2. View the list of all machines/assets
3. Click a machine to view its card (details, status, location)

## Machine Parts

1. Navigate to **Parts**
2. View spare parts catalog

## Machine Documents

1. Navigate to **Documents** (per machine)
2. View uploaded manuals and certificates

## Maintenance Dashboard

1. Navigate to **Dashboard** under Maintenance
2. View KPIs: machines under maintenance, open requests, overdue tasks
3. View upcoming preventive tasks

## Maintenance Requests

1. Navigate to **Requests**
2. View request list with status
3. Track: Open → In Progress → Completed

## Tasks

1. Navigate to **Tasks**
2. View all tasks
3. Click **My Tasks** to see assigned tasks
4. Start and complete tasks as needed

## Preventive Maintenance

1. Navigate to **Preventive** → **Schedules**
2. View upcoming and overdue tasks
3. Calendar view shows scheduled dates

## Downtime

1. Navigate to **Downtime**
2. View downtime events
3. Analysis shows patterns and causes

## Practical Exercise

1. Find a machine and view its card
2. Check the maintenance dashboard KPIs
3. View open requests
4. Check your tasks
5. View upcoming preventive tasks

## Summary

You now know how to:
- Navigate maintenance modules
- View machines and their details
- Track requests and tasks
- Log and analyze downtime
- View preventive schedules
