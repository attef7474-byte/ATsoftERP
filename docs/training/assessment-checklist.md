# Assessment Checklist

> Trainee assessment for ATsoft ERP training

## Session 1: System Overview

- [ ] Can log in successfully
- [ ] Can navigate using the sidebar
- [ ] Can switch between English and Arabic
- [ ] Can use F9 search

## Session 2: Inventory Operations

- [ ] Can view warehouses
- [ ] Can view products and categories
- [ ] Can check inventory balances
- [ ] Can view movement history
- [ ] Understands the purpose of inventory counts

## Session 3: Maintenance Operations

- [ ] Can view machines and their details
- [ ] Can read the maintenance dashboard
- [ ] Can view requests
- [ ] Can view tasks
- [ ] Understands preventive maintenance
- [ ] Understands downtime logging

## Session 4: Barcodes, Reports, Search

- [ ] Can view barcode records
- [ ] Can view and export reports
- [ ] Can export CSV
- [ ] Can export Excel
- [ ] Understands PDF is browser print-to-PDF
- [ ] Can use F9 search effectively

## Session 5: Admin

- [ ] Can list and view users
- [ ] Understands roles and permissions
- [ ] Can start API and Web services
- [ ] Can run health checks
- [ ] Can run smoke tests
- [ ] Understands backup procedures
- [ ] Understands security do's and don'ts

## Overall

- [ ] Understands which modules are available
- [ ] Understands which modules are NOT available
- [ ] Can find help in the user manual
