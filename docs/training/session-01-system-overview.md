# Session 1: System Overview

> Training session for all users — 1 hour

## Objectives

By the end of this session, you will be able to:
- Log in to ATsoft ERP
- Navigate the interface
- Switch between English and Arabic
- Use the F9 search

## Step 1: Access the System

1. Open your web browser
2. Go to: http://localhost:3000
3. You will see the login page

## Step 2: Log In

1. Enter your email (provided by admin)
2. Enter your password
3. Click **Sign In**
4. You should see the Dashboard

## Step 3: Explore the Dashboard

The Dashboard shows:
- Summary cards with key metrics
- KPIs
- Recent operations

## Step 4: Navigate Using the Sidebar

The sidebar on the left contains links to all modules:
- Dashboard
- Alerts
- Notifications
- Settings
- Inventory section
- Maintenance section
- Reports
- Search

Click any link to navigate to that page.

## Step 5: Switch Language

1. Look for the language icon in the top bar
2. Click it to switch between English and Arabic
3. The interface will switch to RTL (right-to-left) for Arabic

## Step 6: Use Search (F9)

1. Press **F9** or **Ctrl+K**
2. Type a search term
3. Results appear grouped by entity type
4. Click a result to open that record

## What Not to Use

The following are **not available** in this release:
- Sales, Purchasing, Finance, HR
- AI, IoT, BI, Forecasting
- Workflows
- Import/Export Designer
- Print Template Designer

## Summary

You now know how to:
- Log in
- Navigate
- Switch language
- Search
