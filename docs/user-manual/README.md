## Quick Links

- **Web app**: http://localhost:3000
- **Login**: admin@atsofterp.com (provided by admin)
- **API docs (Swagger)**: http://localhost:4000/api/docs

## Not in This Release

The following are **not** available in the current release:
- Sales
- Purchasing
- Finance / Accounting
- HR / Employees
- AI / IoT / BI / Forecasting
- Workflows
- Import/Export Designer
- Print Template Designer

## Language

The interface supports both English and Arabic. Use the language switcher in the top bar.
