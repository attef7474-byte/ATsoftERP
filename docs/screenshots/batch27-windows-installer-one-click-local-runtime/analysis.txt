## Design Decisions

- No external dependencies beyond PowerShell 7, Node.js 20+, and sqlcmd
- PID files in C:\ATsoftERP\Temp\ for process tracking
- CMD wrappers for desktop double-click compatibility
- Launcher menu for non-PowerShell users
- Backup-as-precondition for migrations (safe by default)
- -DryRun support on all destructive operations
- Safe uninstall (backups/logs/config preserved unless -RemoveAll)
- Process mode default; service mode opt-in via nssm
