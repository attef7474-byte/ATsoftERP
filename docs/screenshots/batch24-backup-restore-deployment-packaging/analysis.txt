Key Design Decisions
--------------------
- NO Docker references — pure Windows local deployment
- NO PostgreSQL — SQL Server only (sqlserver://localhost:50079)
- Versioned releases with symlink-based rollback (zero-downtime)
- sc.exe for Windows services (lightweight, no nssm dependency)
- All scripts support -DryRun for safe preview
- Restore test always creates a random-suffixed disposable DB
- JWT_SECRET placeholder detection in validate-env
