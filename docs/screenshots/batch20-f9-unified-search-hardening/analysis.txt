warehouseLocationAdapter Limitation
-----------------------------------
The warehouseLocationAdapter endpoint remains at '/inventory/warehouses'
(as inherited from Batch 18). It was not changed to avoid breaking the
barcodes/generate page which uses this adapter with a warehouseId filter.

Permission Filtering Deferred
-----------------------------
Permission-aware entity type filtering (hiding entity types based on
user permissions) is not implemented at the frontend level. The backend
API enforces permissions — if a user lacks read permission for an entity,
the API call fails silently and that entity type is omitted from results.
This is intentional to avoid duplicating permission logic on the frontend.
